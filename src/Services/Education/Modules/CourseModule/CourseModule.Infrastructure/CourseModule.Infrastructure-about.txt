Infrastructure Layer

Ef Configurations Ц A management class for EF modification configurations for Entities in the Domain Layer.

Persistence Ц A class implementing the base context for working with the database. Currently, specifically implemented for EF.

Extensions Ц A folder containing classes adapted for general Dependency Injection (DI).

Repositories Ц Contains the main Unit of Work implementation and implementations of database interaction interfaces for Entities in the Domain Layer.
